# Holberton School — Higher Level Programming

![Holberton](https://img.shields.io/badge/Holberton-School-red?style=flat-square) ![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)

## Description

This repository covers **Python programming** from the fundamentals to object-oriented programming, SQL databases, web scraping, and more. It is one of the core tracks at Holberton School that transitions students from C to higher-level languages.

## Projects

| Directory | Topic |
|-----------|-------|
| `python-hello_world` | Python basics, print, strings |
| `python-if_else_loops_functions` | Control flow, loops, functions |
| `python-import_modules` | Modules and imports |
| `python-data_structures` | Lists, tuples |
| `python-more_data_structures` | Sets, dictionaries |
| `python-exceptions` | Exception handling |
| `python-classes` | OOP — classes, attributes, methods |
| `python-more_classes` | Advanced OOP |
| `python-everything_is_object` | Python object model |
| `python-inheritance` | Inheritance and polymorphism |
| `python-input_output` | File I/O |
| `python-serialization` | JSON serialization |
| `python-test_driven_development` | TDD with doctest and unittest |
| `SQL_introduction` | MySQL basics |
| `SQL_more_queries` | Joins, subqueries, views |
| `javascript-warm_up` | JavaScript basics |
| `javascript_objects_scopes_closures` | JS OOP |

## Learning Objectives

- Write clean, Pythonic code following PEP8
- Understand Python object model and memory management
- Apply OOP concepts: encapsulation, inheritance, polymorphism
- Work with SQL databases using MySQL
- Handle files, JSON serialization, and network requests
